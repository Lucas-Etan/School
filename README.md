# School

# 实验目标

- 理解传统 SaaS 架构与 AI Agent 架构的核心差异
- 掌握 Dense 与 MoE 模型架构的特点和适用场景
- 学会构建原子化业务工具库（Atomic Tools）
- 掌握 MCP 风格的工具 Schema 描述方法，理解它与普通函数调用、完整 MCP Server 的区别
- 理解 Agent 核心调度器的工作原理
- 学会实现多工具链式调用（Chain of Actions）
- 掌握自然语言界面（LUI）的构建方法
- 建立 AI 时代软件工程的新思维
